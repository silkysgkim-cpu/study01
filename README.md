## VSCode에서의 단축키

- 탭 이동 단축키 입니다. 
`command` + `shift`  + `[` / `]`

- 탭 닫기 단축키 입니다.
`command` + `w`

- 새탭 열기 단축키 입니다.
`command` + `shift` + `t`

- 여는 태그와 닫는 태그를 함께 생성해주는 단축키 입니다.
`optoin` + `w`

## 크롬 브라우저에서의 단축키

- 검색 주소창으로 이동하는 단축키 입니다.
`command` + `l`

- 닫았던 창을 여는 단축키 입니다.
`command` + `shift` + `t`

- 새 창을 여는 단축키 입니다.
`command` + `n`

- 새 탭을 여는 단축키 입니다.
`command` + `t`
